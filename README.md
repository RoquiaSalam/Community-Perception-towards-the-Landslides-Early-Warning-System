# Community Perception towards the Landslides Early Warning System

This repository contains R scripts used to conduct structural equation modelling (SEM) for analysing household-level landslide early warning system pathways in Rangamati, Bangladesh.

The analysis examines relationships among landslide risk, landslide forecasting, warning dissemination and communication, warning performance, and preparedness and response.
